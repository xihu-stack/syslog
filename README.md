# IP-Guard + 深信服 员工行为 AI 分析系统

基于**本地大模型**（Qwen3-32B / DeepSeek）分析员工终端行为日志，识别**数据泄露 / 求职离职 / 行为偏离**等风险，近实时告警。

## ✨ 核心功能
- 📊 **监控大屏**：风险意图分布饼图、高风险用户TOP10、合并告警表（按用户聚合）
- 👤 **用户风险画像**：风险趋势折线、活跃时段、行为基线、研判历史、行为时间线
- 🔍 **告警研判**：搜索+风险筛选、AI解释+证据时间线+基线偏离对比
- ⚙️ **后台配置**：多模型LLM（活动+兜底）、Syslog实时接收、字典管理
- 📡 **数据源**：IP-Guard（文档操作/网页浏览/关键字搜索）+ 深信服AC（上网行为管理）

## 🏗️ 技术栈
- 后端：Python 3.12 + FastAPI + SQLAlchemy + SQLite（可切PostgreSQL）
- 前端：React 18 + Ant Design 5 + ECharts 5（BootCDN，免Node构建）
- AI：Qwen3-32B / DeepSeek（本地私有化，OpenAI兼容接口）

## 📐 架构亮点
- **增量研判**：只研判新事件，历史累积不重算
- **基线偏离**：画像=窗口之前的历史，数值化偏离检测
- **降噪聚合**：几万条上网日志→降噪过滤+按域名时间桶聚合→几千条
- **并发安全**：研判不持写锁（WAL+busy_timeout），压测0锁冲突
- **冷启动压制**：无基线时新域名不误报，只有招聘/网盘/远程控制/凌晨等具体信号才提分
